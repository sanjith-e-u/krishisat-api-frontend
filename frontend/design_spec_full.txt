**bg-grid-fade** (hero only):

## Phase 4 — Keyframe animation system

Define these keyframes globally, then expose as Tailwind utilities
(@utility or tailwind.config animation extension, match whatever this
project already uses):

```
fadeUp:      0.7s cubic-bezier(0.16,1,0.3,1) — translateY(12px)→0, opacity 0→1
fadeIn:      0.6s ease-out — opacity only
reveal:      0.7s cubic-bezier(0.16,1,0.3,1) — translateY(16px)→0, used w/ IntersectionObserver
auroraDrift: 18s ease-in-out — translate3d(-8%,-4%)→(8%,6%) + scale 1→1.15
auroraDrift2: 22s ease-in-out — opposite drift + scale(1.1)
gridDrift:   28s linear — background-position 0 0 → 56px 56px
float:       5s ease-in-out — translateY(0)→-6px
marquee:     32s linear — translateX(0)→-50%
spinSlow:    10s-50s linear — rotate(0)→360deg
pulseDot:    2s — opacity 1→0.35→1
pulseRing:   2.4s — box-shadow ring expands 0px→8px then fades
scan:        3.8s — horizontal laser line sweeps -10%→110%
shimmer:     7s — background-position 200%→-200%
holoSweep:   5s — gradient position 0%→200%
```

Map to utility classes: animate-fade-up, animate-fade-in,
animate-pulse-dot, animate-pulse-ring, animate-float, animate-marquee,
animate-grid-drift, animate-aurora-1, animate-aurora-2, animate-scan.

**Scroll progress bar**: fixed, 2px tall, top of page, z-[60]. Width


**Critical — accessibility override** (implement this now, don't

## Phase 5 — Interactive effects (likely needs framer-motion if not
## already installed — confirm from Phase 0 audit first)

**Card shine sweep** (.shine-host + .shine-overlay): on hover, a
skewed gradient band sweeps across — transparent 30% → accent@22%
opacity 50% → transparent 70%, from translateX(-120%) skewX(-20deg)
to translateX(220%) skewX(-20deg), 0.9s cubic-bezier(0.16,1,0.3,1).

**Hover glow** (.hover-glow): lift -2px with layered box-shadow:
```css
box-shadow:
  0 0 0 1px color-mix(in oklab, var(--accent) 40%, transparent),
  0 24px 60px -28px color-mix(in oklab, var(--accent) 55%, transparent);
```

**Magnetic buttons** (.btn-magnetic): hover translateY(-1px) + accent
shadow, active translateY(0), transition 0.25s cubic-bezier(0.16,1,0.3,1).

**3D tilt wrapper** (<Tilt> component): perspective 900px container,
mousemove calculates normalized cursor position (-0.5 to 0.5), rotates
card up to ±6deg on X/Y via rotateY/rotateX, updates --mx/--my CSS
vars for glow overlay, resets smoothly on mouseleave.

**Mouse spotlight** (<MouseSpotlight>): radial gradient overlay
tracking cursor via mousemove listener:
```css
radial-gradient(420px circle at var(--mx,50%) var(--my,0%),
  color-mix(in oklab, var(--accent) 22%, transparent), transparent 60%);
```

**Tilt glow** (.tilt-glow::after): uses --mx/--my from Tilt component,
internal radial glow, opacity 0→1 on hover.

**Gradient ring** (.gradient-ring::before): spinning conic-gradient
border via padding+mask-composite border hack, spinSlow 10s linear
infinite, opacity 0.65.
